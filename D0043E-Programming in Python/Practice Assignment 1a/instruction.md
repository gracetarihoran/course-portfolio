#### Mini-Project: Rectangle and Circle Area Calculations

**Learning Goals**

*   Understand how to declare and assign values to variables in Python.
*   Learn to use formulas for basic geometric area calculations.
*   Practice writing Python scripts and outputting results in a formatted string.

**Task**

1.  In a file named `answer.py` complete the following tasks.
    
2.  **Rectangle Area Calculation**:
    
    *   Declare two variables: `length` and `width`.
    *   Assign the value `10` to `length` and `33` to `width`.
    *   Calculate the area of the rectangle, `rectangle_area`, using the formula: `area = length * width`.
    *   Print the result in the exact format:
        
            The area of the rectangle is: [result]
        
3.  **Circle Area Calculation**:
    
    *   Declare two variables: `PI` and `radius`.
    *   Assign the value `3.14` to `PI` and `5` to `radius`.
    *   Calculate the area of the circle, `circle_area`, using the formula: `area = PI * (radius ** 2)`.
    *   Print the result in the exact format:
        
            The area of the circle is: [result]
