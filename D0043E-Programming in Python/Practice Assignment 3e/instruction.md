**Concepts Covered:** Defining and calling functions, parameters, `return`

**Description:**

Create a program that converts temperatures between Celsius and Fahrenheit.

*   Write two functions:
    *   `celsius_to_fahrenheit(celsius)` to convert Celsius to Fahrenheit.
    *   `fahrenheit_to_celsius(fahrenheit)` to convert Fahrenheit to Celsius.
*   Ask the user which conversion they want to perform and the input temperature. Ensure the user inputs valid values, using a while loop. For example, keep asking user to enter a value until it is either C or F
*   Display the result using the appropriate function.

**Example Interaction:**

    Enter 'C' to convert Celsius to Fahrenheit, or 'F' to convert Fahrenheit to Celsius: CEnter the temperature: 2525.0°C is 77.00°F
