**Mini-Project: Grade Calculator with Report**
----------------------------------------------

***

### **Learning Goals**

*   Understand how to receive and validate user input in Python.
*   Practice arithmetic operations to calculate averages.
*   Learn how to use conditionals (`if`, `elif`, and `else`) to evaluate and assign grades.
*   Develop formatted output to display results clearly and professionally.

***

### **Task**

In a file named `grade_calculator.py`, complete the following tasks:

***

### **User Input and Validation**

1.  Ask the user to input scores for **5 subjects**.

    "Enter the score for Subject 1: ""Enter the score for Subject 2: ""Enter the score for Subject 3: ""Enter the score for Subject 4: ""Enter the score for Subject 5: "

***

### **Calculate Average**

1.  Compute the average of the five subject scores using the formula:
    
        average = (subject1 + subject2 + subject3 + subject4 + subject5) / 5
    

***

### **Assign Grades**

*   Use the following grading scale to determine the final grade:
    *   **90–100:** A
    *   **80–89:** B
    *   **70–79:** C
    *   **60–69:** D
    *   **Below 60:** F

***

### **Display the Grade Report**

*   Print the scores, average, and final grade in the **exact format** below:

    Grade Report----------------------Subject 1: [score1]Subject 2: [score2]Subject 3: [score3]Subject 4: [score4]Subject 5: [score5]Average Score: [average]Final Grade: [grade]
