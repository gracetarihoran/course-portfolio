Assignment description
Concepts Covered: while loops, break, continue

Description:

Create a number guessing game where the user has 5 chances to guess a random number.

The game should repeatedly prompt the user for a guess until they either guess correctly or use up all their attempts.
If the user provides an invalid input, handle it appropriately. It is not considered a guess if the user enters an invalid entry (so don't increase the guess count for invalid entries). For example, Invalid entries are some random string, a number out of range of 1 to 20.
Provide feedback after each guess.
If the user uses up all their attempts, print "Sorry, you've run out of attempts. The correct number was 15." Where 15 should be replaced by the randomly generated number.
Example Interaction:

Guess a number between 1 and 20: 25
Out of range! Guess again.
Guess a number between 1 and 20: 10
Too low!
Guess a number between 1 and 20: 15
Correct! You guessed it in 3 attempts.
